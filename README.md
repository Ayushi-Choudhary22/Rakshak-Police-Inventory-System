# 🚨 Rakshak - Police Inventory Management System

## 📌 Overview
Rakshak is a full-stack Police Inventory Management System designed to digitize and streamline the management of police resources, station data, and operational records. It enables efficient tracking, allocation, and monitoring of inventory across departments, improving transparency and operational efficiency.

The system is built with a modern web stack and integrates Firebase for authentication and backend services.

---

## 🌐 Live Demo
👉 https://rakshak-police-inventory-system.vercel.app

---

## ✨ Features
- Secure authentication system (Firebase Auth)
- Department-wise dashboard
- Station management system
- GPS/location tracking module
- Real-time data handling
- Role-based access system
- Responsive UI for all devices

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** Firebase Firestore
- **Authentication:** Firebase Authentication
- **Deployment:** Vercel
