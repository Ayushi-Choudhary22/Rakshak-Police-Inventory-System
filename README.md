# 🛡️ Rakshak – Police Inventory Management System

---

## 📌 Overview
Rakshak is a full-stack web application designed to digitize and streamline police inventory management. It enables centralized tracking of police station resources, equipment, and departmental assets.

The system improves transparency, reduces manual errors, and provides efficient digital management of inventory across departments.
